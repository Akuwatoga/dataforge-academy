# Projects｜项目制实战

本目录用于承载 DataForge Academy 的项目制实战内容。

## 项目列表

| 项目 | 主题 | 目标 |
|---|---|---|
| project-01-python-sql-etl | Python + SQL + ETL 入门 | 完成数据生成、清洗、入库和指标分析 |
| project-02-ecommerce-data-warehouse | 电商数仓建模 | 完成 ODS/DWD/DWS/ADS 分层和指标体系 |
| project-03-flink-cdc-starrocks | Flink CDC 实时同步 | MySQL binlog 实时同步到 StarRocks/Doris |
| project-04-kafka-realtime-analytics | Kafka 实时行为分析 | 用户行为日志实时采集和分析 |
| project-05-olap-optimization | OLAP 查询优化 | 完成分区、分桶、物化视图、Profile 优化案例 |
| project-06-ai-data-report | AI 数据报告 | 构建数据看板和 AI 自动经营日报 |

## 项目文档标准

每个项目目录应包含：

```text
README.md           # 项目说明
requirements.md     # 环境和依赖
architecture.md     # 架构设计
steps.md            # 实操步骤
knowledge.md        # 涉及知识点
interview.md        # 面试可讲点
```
