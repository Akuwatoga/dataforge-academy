# Tracks｜分方向学习路径

本目录用于承载不同职业方向的学习路径。

## 路径列表

| 路径 | 说明 |
|---|---|
| 01-data-foundation | Python、SQL、数据库、Linux、Docker 基础 |
| 02-etl-engineer | ETL、数据同步、数据清洗、数据质量 |
| 03-data-warehouse | 数仓分层、维度建模、指标体系 |
| 04-big-data-engineer | Kafka、Flink、Spark、Hive、大数据平台 |
| 05-realtime-warehouse | Flink CDC、实时指标、StarRocks/Doris |
| 06-olap-engineer | OLAP 引擎、表模型、查询优化 |
| 07-ai-data-product | 数据看板、AI 报告、数据问答、数据产品 |
| 08-career-interview | 简历、面试、项目讲解、场景题 |

## 使用方式

学习者可以从 `LEARNING_PATH.md` 选择适合自己的路径，然后进入对应目录学习。

每条路径后续会包含：

- path.md：完整路线
- knowledge-map.md：知识点地图
- tasks.md：阶段任务
- resources.md：资料和工具
- interview.md：面试重点
