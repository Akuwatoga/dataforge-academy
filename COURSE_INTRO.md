# 课程介绍｜DataForge Academy

## 课程定位

DataForge Academy 是一个面向数据岗位的项目制学习体系，覆盖数据工程、ETL、大数据、实时数仓、OLAP 分析、数据产品和 AI 数据应用。

它不是按照单一技术栈组织的传统课程，而是以真实岗位能力和完整项目链路为主线，将基础知识、工程实践、业务理解、性能优化和求职表达组合在一起。

## 课程目标

学习者最终应能完成一个端到端数据平台项目，并具备以下能力：

1. 使用 Python 生成、清洗和处理数据。
2. 使用 SQL 完成业务指标分析。
3. 设计 MySQL 业务库和基础数据模型。
4. 使用 Docker 和 Linux 部署数据组件。
5. 使用 ETL / CDC 工具完成数据同步。
6. 使用 Kafka 和 Flink 构建实时数据链路。
7. 使用 StarRocks / Doris 构建 OLAP 分析数仓。
8. 设计 ODS、DWD、DWS、ADS 数仓分层。
9. 完成常见 SQL 性能优化案例。
10. 构建数据看板和 AI 自动分析报告。
11. 将项目包装成简历、面试和作品集材料。

## 适合人群

- 计算机、数据、管理科学、信息系统等相关专业学生
- 想转向 ETL、数据工程、大数据工程、实时数仓方向的人
- 有 Python / SQL 基础但缺少完整项目的人
- 想在 gap 期间系统补齐数据能力的人
- 想用 GitHub 作品集提高求职竞争力的人
- 想长期从事数据方向教培、辅导或内容创作的人

## 学习方式

本课程采用“项目驱动 + 每日输出 + 阶段复盘”的方式。

每个阶段都包含：

1. 明确目标：本阶段解决什么问题。
2. 知识点学习：学习完成任务必须掌握的概念。
3. 项目实操：在真实项目中使用这些知识。
4. 文档沉淀：记录设计、踩坑、命令、SQL 和结论。
5. 视频录制：将学习过程转化为可发布内容。
6. 面试转化：整理成简历项目和面试表达。

## 主线项目

主线项目为：**面向电商业务的端到端数据平台项目**。

业务场景包括：

- 用户注册
- 商品浏览
- 加购
- 下单
- 支付
- 退款
- 渠道投放
- 商品销售
- 用户留存
- 实时经营分析

技术链路包括：

```text
Python 模拟数据
MySQL 业务库
Kafka 行为日志
Flink CDC / Flink Streaming
StarRocks / Doris OLAP 数仓
ODS / DWD / DWS / ADS 分层
SQL 指标分析
物化视图与查询优化
Streamlit / Superset 看板
AI 自动经营日报
```

## 课程模块

| 模块 | 主题 | 目标 |
|---|---|---|
| Module 00 | 项目定位与学习方法 | 明确岗位能力、项目结构和输出方式 |
| Module 01 | Python + SQL 基础 | 补齐数据岗位基本功 |
| Module 02 | MySQL 业务库 | 理解 OLTP 业务建模 |
| Module 03 | ETL 与数据质量 | 完成批处理数据链路 |
| Module 04 | Docker + Linux | 掌握服务器部署能力 |
| Module 05 | 数据仓库建模 | 掌握 ODS/DWD/DWS/ADS |
| Module 06 | Kafka + Flink | 构建实时数据处理链路 |
| Module 07 | StarRocks / Doris | 构建实时 OLAP 数仓 |
| Module 08 | OLAP 优化 | 完成慢查询优化案例 |
| Module 09 | 数据产品与 AI 报表 | 做出看板和 AI 分析报告 |
| Module 10 | 简历与面试 | 将项目转化为职业竞争力 |
